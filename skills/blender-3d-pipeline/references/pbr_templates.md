# PBR Material Templates

Common material presets for game assets.

## Metal Materials

### Polished Steel

```bash
--color 0.6,0.6,0.65 --metallic 1.0 --roughness 0.1
```

### Brushed Aluminum

```bash
--color 0.9,0.9,0.92 --metallic 1.0 --roughness 0.4
```

### Gold

```bash
--color 1.0,0.766,0.336 --metallic 1.0 --roughness 0.2
```

### Copper

```bash
--color 0.955,0.637,0.538 --metallic 1.0 --roughness 0.25
```

## Non-Metal Materials

### Plastic (Red)

```bash
--color 0.8,0.1,0.1 --metallic 0.0 --roughness 0.3
```

### Rubber

```bash
--color 0.1,0.1,0.1 --metallic 0.0 --roughness 0.9
```

### Wood

```bash
--color 0.4,0.25,0.1 --metallic 0.0 --roughness 0.7
```

### Concrete

```bash
--color 0.5,0.5,0.5 --metallic 0.0 --roughness 0.95
```

## Stylized Materials

### Cartoon Outline

```bash
--color 0.9,0.9,0.9 --metallic 0.0 --roughness 1.0
```

### Emissive Glow

```bash
--color 0.2,0.5,1.0 --emission-color 0.2,0.5,1.0 --emission-strength 5.0
```
