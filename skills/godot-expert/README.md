# Godot Expert Skill

A comprehensive Claude Code skill for Godot 4.x game development. Provides patterns, best practices, and ready-to-use code examples covering the full game development workflow from project setup through multi-platform export.

## When to Use

Activate this skill when working on any Godot Engine project, including:

- Setting up new Godot 4.x projects with proper structure and conventions
- Writing GDScript with type hints, signals, exports, and async/await
- Implementing physics, collision, and input handling (2D and 3D)
- Building UI with Control nodes, themes, and HUD patterns
- Configuring animation (AnimationPlayer, AnimationTree, Tweens) and audio
- Optimizing performance with object pooling, LOD, and profiling
- Exporting to HTML5, Android, iOS, and desktop platforms
- Writing tests with the GDUnit framework

## Topics Covered

1. **Project Structure** - Standard layout, project.godot config, naming conventions
2. **Scenes & Nodes** - Composition patterns, instancing, inheritance, groups
3. **GDScript Patterns** - Type hints, exports, signals, onready, async/await, custom resources, singletons
4. **Physics & Collision** - CharacterBody2D/3D movement, collision layers, Area2D detection, raycasts
5. **Input Handling** - Input actions, events, touch/mobile controls
6. **UI/Control Nodes** - Scene structure, responsive UI, themes, HUD
7. **Animation & Audio** - AnimationPlayer, AnimationTree, tweens, audio manager
8. **Performance** - Object pooling, LOD, profiling
9. **Export Targets** - Multi-platform export overview
10. **Testing** - GDUnit test structure and patterns
